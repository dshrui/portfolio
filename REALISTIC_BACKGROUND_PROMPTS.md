# Realistic Background Prompts

Use these prompts when regenerating the three SME adaptation backgrounds. Export each result as a JPG in the matching file path below.

Recommended settings:

- Aspect ratio: 4:5 portrait
- Target size: 1080 x 1350 px if available
- Style: photorealistic commercial/editorial photography
- Direction: campaign-specific AI image in the same logic as Chirodayz and Padel Revolution, where the image explains the post topic and is not just decoration
- Avoid generated text inside the image because the HTML/CSS already overlays the poster copy
- Avoid people, hands, logos, labels, watermarks, and readable words
